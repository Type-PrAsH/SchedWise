AI Student Free-Time Optimizer
Overview

Students often lose valuable time due to timetable gaps, cancelled classes, or unplanned free slots. This project addresses that problem by converting unused free time into structured, meaningful learning opportunities using AI.

The system analyzes a student’s available free slots and selected skills, then intelligently suggests focused activities and relevant learning resources. Instead of procrastination, students get clarity on what to do, when to do it, and why it matters.

Key Features

Free Slot Detection – Identifies available time gaps in a student’s schedule

AI-Powered Suggestions – Recommends activities based on slot duration and skill priorities

Skill-Focused Learning – Ensures suggestions align strictly with chosen skills

Smart Resource Mapping – Suggests learning content (e.g., videos) matching available time

Minimal & Student-Centric Design – Simple, fast, and distraction-free experience

Tech Stack

Frontend: React + TypeScript

Styling: Tailwind CSS

AEngine: Google Gemini API

Authentication: Firebase Authentication

Hosting: Vercel

How It Works

The student provides their timetable (manual input or upload).

The system identifies free time slots.

The student selects skills they want to improve.

AI analyzes time duration + skill priority.

Focused learning suggestions are generated for each free slot.

Problem Solved

Unproductive use of free time

Learning discontinuity between classes

Lack of structured guidance during short time gaps

This project ensures every free minute has purpose.
